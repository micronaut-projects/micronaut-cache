To use https://ignite.apache.org/[Ehcache] as the caching implementation, add it as a dependency to your application:

dependency:io.micronaut.cache:micronaut-cache-ignite:{version}[]

To have Micronaut create a cache, the minimum configuration is:
[source,yaml]
---
ignite:
  clients:
    default:
      addresses: [ 'ignite.com' ]
  caches:
    my-cache:
      client: default
---

The full list of configurable options via Micronaut properties is below.

include::{includedir}configurationProperties/io.micronaut.cache.ignite.configuration.IgniteCacheConfiguration.adoc[]
include::{includedir}configurationProperties/io.micronaut.cache.ignite.configuration.IgniteClientConfiguration.adoc[]

To disable Ignite:

[source,yaml]
----
ignite:
  enabled: false
----
